# pf_carousel

A Carousel Challenge using vanilla javascript

##Getting Started
There are two methods for getting started with this repo.

##Using Git
    > git clone https://github.com/wastefactory/pf_carousel.git
    > cd pf_carousel
    > npm install
    > npm start
    > go to http://localhost:8080/

##Downloading Zip
    > cd pf_carousel
    > npm install
    > npm start
    > go to http://localhost:8080/
